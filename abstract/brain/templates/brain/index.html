Создание конспекта на основе текста Веб-приложение, которое будет генерировать краткий конспект для заданного текста. Конспект будет представлять из себя набор предложений текста, которые имеют наибольшую близость с ключевыми словами. Ключевые слова вычленеяются по средствам обученной модели word2vec. Реализация приложения предполагется на основе Django и должна включать в себя базовый функцонал по загрузке текста и вывода конспекта для этого текста.
